Physical Tags

Physical tags are used to control how text and other content should be presented in a browser. Physical tags are used sparingly in modern web development, but they're good to have in your back pocket. Many physical tags are only for visual web browsers.

Font Tag

The most recognizeable physical tag is the font tag. You can use the font tag to control many aspects of the presentation of text.

Underline, Italics, Bold, and Strikethrough Tags

You can also underline, italicize, bold, and strike through your text with physical formatting tags.

Break Tag

Superscript and Subscript

The superscript and subscript tags are other good uses of physical formatting tags, though many non-visual browsers make use of them as logical tags as well.
