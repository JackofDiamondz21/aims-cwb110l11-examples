Logical Tags

Logical tags are used to describe the meaning of the text/media that the tags enclose. However, most visual browsers will also change the appearance of logical tags.

Header Tags

The h1, h2, h3, h4, h5, and h6 tags define headers and subheaders of a document. Often times, visual browsers will decrease the font size of header tags as the header number increases. For example, an h6 tag is smaller than an h4 tag.

Paragraph

The paragraph tag indicates that the text enclosed is a paragraph of text.

Emphasis

The emphasis tag gives emphasis to the enclosed text. Often times, visual browsers italicize the text within these tags.

Strong

The strong tag gives an even stronger emphasis to the enclosed text. Often times, visual browsers also bold the text within these tags.

Div tags are a logical tag that divide sections of content.
