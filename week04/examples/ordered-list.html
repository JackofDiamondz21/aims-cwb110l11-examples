Ordered List

Ordered lists are for listing information in a particular order.

  1. First list item
  2. Second list item
  3. Third list item
  4. Fourth list item
  5. Fifth list item

You can start a list at any number you specify via the start attribute.

  1. Tenth list item
  2. Eleventh list item
  3. Twelfth list item
  4. Thirteenth list item
  5. Fourteenth list item

You can also reverse a list:

  1. Tenth list item
  2. Ninth list item
  3. Eigth list item
  4. Seventh list item
  5. Sixth list item

You can also specify the type of incrementation used:

  1. Tenth list item
  2. Ninth list item
  3. Eigth list item
  4. Seventh list item
  5. Sixth list item
